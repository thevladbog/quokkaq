# Интеграции

Подключение внешних сервисов, которые использует ваша организация.

## Календарь

- **[Google Календарь](./google-calendar)** — вход через Google и выбор календаря с правом записи для подразделения.
- **[Яндекс.Календарь (CalDAV)](./yandex-calendar)** — CalDAV, логин Яндекса и пароль приложения.

## Поддержка и обращения

- **[Яндекс.Трекер](./yandex-tracker)** — OAuth или IAM, организация и очередь для обращений из QuokkaQ.
- **[Plane (self-hosted)](./plane)** — развёртывание Plane и переменные окружения (опционально; можно отключить через env).

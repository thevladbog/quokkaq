# Integrations

Connect external services used by your organization.

## Calendar

- **[Google Calendar](google-calendar.md)** — sign in with Google and pick a writable calendar per unit.
- **[Yandex Calendar (CalDAV)](yandex-calendar.md)** — CalDAV URL, Yandex login, and app password.

## Support and tickets

- **[Yandex Tracker](yandex-tracker.md)** — OAuth or IAM, organization id, queue key for QuokkaQ support reports.
- **[Plane (self-hosted)](plane.md)** — deploying Plane and environment variables for QuokkaQ support reports (optional; can be disabled via env).

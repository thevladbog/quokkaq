# Интеграции

Подключение внешних сервисов, которые использует ваша организация.

## Календарь

- **[Google Календарь](google-calendar.md)** — вход через Google и выбор календаря с правом записи для подразделения.
- **[Яндекс.Календарь (CalDAV)](yandex-calendar.md)** — CalDAV, логин Яндекса и пароль приложения.

## Поддержка и обращения

- **[Яндекс.Трекер](yandex-tracker.md)** — OAuth или IAM, организация и очередь для обращений из QuokkaQ.
- **[Plane (self-hosted)](plane.md)** — развёртывание Plane и переменные окружения (опционально; можно отключить через env).
